# AI 助手

想要开发自己的多人在线游戏却不知从何下手？借助 Tiaoom 提供的 **Agent Skills**，即使你不是资深开发者，也能在 AI 助手的帮助下快速构建出专业级的游戏。

## 什么是 Agent Skills？

Agent Skills 就像是给 AI 助手（如 Claude Code、GitHub Copilot、Cursor、Gemini CLI）的一本“开发者秘籍”。我们把 Tiaoom 平台的架构、API、开发规范和最佳实践全部打包成了标准化的技能文件，存放在项目的 `game/skills` 目录中。

当你把这些文件交给 AI 时，它会瞬间变成一个“Tiaoom 开发专家”，为你提供精准的代码建议。

## 对你有什么好处？

*   **零门槛启动**：不需要先啃完所有文档，AI 会告诉你第一步该做什么。
*   **代码秒生成**：直接生成符合平台规范的后端逻辑和前端界面，减少手写错误。
*   **即时专家指导**：遇到报错或逻辑卡壳？AI 会查阅内置的故障排除指南帮你一键修复。
*   **标准统一**：生成的代码自带类型定义和最佳实践，让你的游戏更易维护。

## 如何开始使用？

你可以通过以下三步，让 AI 助手带你飞：

### 第一步：准备好你的 AI 工具
推荐使用支持项目 Agent Skills 的 AI 工具，如 **GitHub Copilot Chat**、**Cursor** 或 **Claude Code**。

### 第二步：让 AI 加载“秘籍”
在对话框中，你可以直接告诉 AI 去阅读我们的技能入口文件。你可以尝试这样说：

> "我想开发一个简单的猜拳游戏，请先阅读 `game/skills/SKILL.md` 了解开发标准，然后告诉我该如何开始。"

### 第三步：按需获取详细指南
我们的技能是模块化的，你可以引导 AI 深入研究特定领域：

**场景 1：询问如何开始开发游戏**
```
我想为 Tiaoom 平台开发一个井字棋游戏，请问我需要做什么？
```

**场景 2：询问具体功能实现**
```
如何在游戏中实现倒计时功能？玩家超时后自动跳过回合。
```

**场景 3：请求代码示例**
```
帮我写一个石头剪刀布游戏的后端代码框架。
```

**场景 4：调试问题咨询**
```
我的游戏在玩家断线重连后状态丢失了，应该如何解决？
```

**场景 5：询问最佳实践**
```
在 Tiaoom 游戏开发中，如何正确处理游戏结束时的积分奖励？
```

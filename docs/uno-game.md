# UNO 游戏使用指南

## 游戏概述

UNO 是一个经典的纸牌游戏，玩家需要匹配颜色或数字来出牌，第一个出完所有牌的玩家获胜。

## 游戏规则

### 基本规则
- 每个玩家开始时有 7 张牌
- 玩家需要出与顶部弃牌相同颜色或数字的牌
- 如果无法出牌，必须从牌堆抽牌
- 当剩下最后一张牌时，必须喊 "UNO!"

### 特殊牌效果

#### 数字牌 (0-9)
- 普通牌，只需匹配颜色或数字

#### 功能牌
- **跳过 (Skip)**: 下一个玩家失去回合
- **反转 (Reverse)**: 改变出牌方向
- **+2**: 下一个玩家抽 2 张牌并失去回合

#### 万能牌
- **变色 (Wild)**: 可以改变当前颜色
- **+4 (Wild Draw Four)**: 可以改变颜色，下一个玩家抽 4 张牌

## 操作说明

### 创建房间
1. 在游戏大厅选择 "UNO" 游戏类型
2. 设置房间名称和密码（可选）
3. 选择玩家人数（2-4人）
4. 点击创建房间

### 游戏操作
1. **出牌**: 点击可以出的牌（会高亮显示）
2. **抽牌**: 点击"抽牌"按钮
3. **选择颜色**: 出万能牌时会提示选择颜色
4. **喊 UNO**: 当剩最后一张牌时点击 "UNO!" 按钮

### 游戏界面
- **顶部**: 显示当前颜色和累积抽牌数
- **中间**: 显示弃牌堆和牌堆
- **其他玩家区域**: 显示其他玩家的牌数和当前回合
- **底部**: 显示你的手牌和操作按钮

## 获胜条件

第一个出完所有手牌的玩家获胜！

## 技术实现

### 后端实现
- 位置: `game/backend/src/games/uno.ts`
- 功能: 游戏逻辑、牌组管理、回合控制

### 前端实现
- 主组件: `game/frontend/src/components/uno/UnoRoom.vue`
- 卡牌组件: `game/frontend/src/components/uno/UnoCard.vue`
- 功能: 游戏界面、交互逻辑、实时同步

### 游戏状态管理
- 使用 Socket.IO 进行实时通信
- 游戏状态在服务端统一管理
- 支持多玩家并发游戏
